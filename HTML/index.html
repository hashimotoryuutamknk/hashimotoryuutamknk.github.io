URIの表記について

"http" scheme の authority にリンクを貼る（と言えばいいの？）。
そのときの表記は， href="http://www.kagawa-nct.ac.jp"とすべきか。 href="http://www.kagawa-nct.ac.jp/"とすべきか。

RFC3986の 6.2.3節の記述によると，

どちらでも同じであるが， "http" scheme に対しての normal form は後者

ということらしい。

そんなわけで， href="http://www.kagawa-nct.ac.jp/"とする方がよさそう。

プログラムのコード

HTML 4.01 Specification の 9章 Text の記述を参考にすると， 以下のような具合か。

  • プログラムのコードの断片は <code>fragment of code</code>。
  • 入力は <kbd>input</kbd>。
  • 出力は <samp>output</samp>。
  • プログラムのソースコードは <pre>source code</pre>。
    あるいは，<pre> element の中で <code> element を使用。
